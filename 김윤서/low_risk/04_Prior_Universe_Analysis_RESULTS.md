## Part 1 (Cell 3) — p/T 비율 진단: 공분산 추정 품질

**목적**: BL의 핵심인 공분산 행렬 Σ(p×p) 추정 품질은 p/T 비율에 직접 의존. 본 프로젝트는 일별 수익률로 Σ 추정 → **T = TRAIN_WINDOW × 21 = 1260일** 기준으로 진단.

**실행 결과**

```
=== p/T 비율 진단 (T = 1260일, 일별 수익률 기준) ===
       N     p/T        등급
------------------------------
      50    0.04        양호
     100    0.08        양호
     200    0.16        양호
     300    0.24        양호
     400    0.32        양호
     500    0.40        양호

현재 전체 유니버스 평균 종목수 ≈ 408개 → p/T = 0.32 (양호)

[참고] 월별 T=60 기준이라면:
  N= 50: p/T=0.83
  N=100: p/T=1.67  ← 매우 불안정
  N=500: p/T=8.33  ← 매우 불안정
→ 일별 데이터 사용으로 N=500 수준까지도 양호 범위로 진입.
```

### 해석

| p/T 범위 | 등급 | 의미 |
|----------|------|------|
| < 0.5 | 양호 | Ledoit-Wolf 안정 수렴 |
| 0.5 ~ 1.0 | 주의 | 추정 분산 증가 |
| > 1.0 | 불안정 | singular 위험, shrinkage 수렴 보장 불가 |

**일별 Σ 사용의 결정적 효과**:
- 월별 T=60 기준이었다면 N=50만 되어도 p/T=0.83(주의), N=500은 p/T=8.33(심각)
- **일별 T=1260으로 전환하면 N=500도 p/T=0.40(양호)**
- 패널의 월평균 ~410개 종목을 그대로 사용해도 추정 안정성 보장

> **Σ_monthly ≈ 21 × Σ_daily 스케일링**은 IID 근사 하의 단위 변환. ARCH 효과로 위기 시 약간의 편향 가능, S&P 500 대형주 위주에서는 작다고 가정.

### 설계 결정

**일별 수익률 사용 + 패널 그대로 (월평균 ~410)** 채택. p/T 측면에서 안정 보장.

### final과의 관계

- final/`bl_functions.py`의 `compute_sigma`도 일별 Ledoit-Wolf × 21 스케일링 동일 사용
- final이 N=410 그대로 19개 실험 모두 안정 실행 → **p/T 진단의 권고가 정확히 적중**
- final에 명시적 p/T 분석은 없음 → **04의 진짜 부가가치**

## Part 2 (Cell 5) — Prior 비교: π(균형 수익률) 분포

**목적**: BL 모형에서 π = λΣw_mkt는 뷰(Q)를 반영하기 전의 기본 기대수익률. Prior w_mkt(시총 vs 1/N) 선택이 실제로 유의미한지 확인.

**실행 결과**

```
=== Prior 비교 수치 요약 (시가총액 vs 1/N) ===
시가총액 Prior HHI 평균:  0.01672
1/N Prior HHI 평균:       0.00268
HHI 비율 (mcap/equal):    6.2x       ← 시가총액 prior가 6배 더 집중

π 표준편차 평균 (mcap):   0.00375
π 표준편차 평균 (1/N):    0.00415
π 상관계수 평균:          0.9865     ← 방향성 거의 동일
```

### 3-패널 그래프 자세한 해석

**① Prior HHI 집중도 (좌)**
- 시가총액 Prior HHI: 0.010~0.021 (시기별 변동, 평균 ~0.017)
- 1/N Prior HHI: 0.0025 근처 평탄
- 비율: 시가총액이 1/N 대비 약 6~8배 집중

> **HHI 해석**: HHI = Σwᵢ². 1/N이면 정확히 1/N. 우리 유니버스 ~410개라 이론적 1/N HHI = 1/410 ≈ 0.00244. 그래프의 1/N(0.0025)과 일치.
>
> 시가총액 HHI 0.017의 **유효 종목 수**(1/HHI) ≈ **59개**. 즉 시가총액 prior는 명목상 410종목에 분산된 것처럼 보이지만 실질적으로 ~60개 메가캡에 비중이 쏠림. **상위 10개가 시총의 ~40%**를 차지하는 미국 시장 구조 반영.

**② π 표준편차 — 균형수익률 분산도 (중앙)**
- 두 prior 라인이 거의 겹침 (시각적으로 구분 어려움)
- 2014년, 2022년에 spike — 시장 변동성 확대 시기 (taper tantrum, 인플레이션 충격)
- 평소 0.001~0.005, 충격기 0.009까지

> **의미**: π = λΣw 이므로 π의 분산도는 prior 형태(w_mcap vs 1/N)보다 **공분산 Σ의 변화**에 더 의존. 즉 "prior를 바꿔도 π의 절대 수준은 비슷하게 흔들린다" — prior 선택의 영향이 제한적임을 시사.

**③ π 상관계수 — 두 prior의 방향성 일치도 (우)**
- 항상 **0.97~1.00** (corr=0.90 기준선보다 훨씬 위)
- 시간 무관 일정

> **결정적 함의**: prior를 mcap에서 1/N으로 바꿔도 **π의 방향성은 거의 동일**. 즉 어떤 종목이 상대적으로 "고평가/저평가" 신호인지 두 prior가 비슷하게 알려줌. 다만 *절대 수준*은 다름 (HHI 차이 6~8배). → **선택의 효과는 "어떤 신호인가"가 아니라 "얼마나 집중할 것인가"** 에 있음.

**3-패널 종합**: ①·③으로부터 prior 변경(mcap → 1/N)은 신호 방향 손실 없이 분산도만 얻을 수 있다. ②는 prior 형태와 무관하게 π가 시장 충격에 동조함을 보여주며, 이는 prior 선택의 영향이 제한적임을 뒷받침한다.

### final 백테스트로 정확히 검증됨

```
final/99_run 백테스트 (2010-2024):
  baseline (mcap):  Sharpe 1.106
  prior_eq (1/N):   Sharpe 1.105
  차이:             0.001  (통계적 노이즈)
```

→ **04의 "두 prior 사실상 동등" 예측이 portfolio 단계에서 정확히 적중**.

### 설계 결정 — Prior 1/N 권고 (유효한 이유)

성과 차이가 0.001로 무의미하다면, 선택은 **맥락**으로 정해짐:

| 1/N 선택 근거 | 설명 |
|---|---|
| ✓ 저위험 anomaly 가설과 일관 | 가설 자체가 "균등 투자 시 저위험 우위" → 1/N이 이를 진술 조건과 일치 |
| ✓ Selection bias 중립 | yfinance 시총 분포가 진짜 시장 시총과 다를 수 있음 (역사적 데이터 한계) |
| ✓ 메가캡 60개 집중 회피 | 시총 prior는 실질 유효 종목 ~60개 → 분산도 손실 |
| ✓ SPLV 등 실제 저위험 ETF도 1/N 사용 | 실무 표준과도 일치 |

**발표/리포트 권고 문구**:
> "Part 2 EDA에서 π 상관 0.9865 발견 — 두 prior가 사실상 동등할 것을 사전 예측. final 백테스트 결과(baseline Sharpe 1.106 vs prior_eq 1.105, 차이 0.001)가 이를 정량적으로 검증. 본 프로젝트는 *저위험 anomaly 활용 ETF* 컨셉이므로 selection bias 중립성 + 가설 일관성 근거로 1/N prior 채택."

### EDA 한계

- 본 분석은 두 prior의 *수치적 차이* 측정엔 충분
- 1/N의 "더 좋다"는 결정적 증거는 아님 — final 결과(0.001 차이)도 어느 쪽이 우수한지 결정 못함
- 둘 다 valid한 선택이며, 선택 기준은 *프로젝트 맥락*으로 결정됨
