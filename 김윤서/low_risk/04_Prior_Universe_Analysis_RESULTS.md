## Part 1 (Cell 3) — p/T 비율 진단: 공분산 추정 품질

**목적**: BL의 핵심인 공분산 행렬 Σ(p×p) 추정 품질은 p/T 비율에 직접 의존. 본 프로젝트는 일별 수익률로 Σ 추정 → **T = TRAIN_WINDOW × 21 = 1260일** 기준.

**실행 결과**

```
=== p/T 비율 진단 (T = 1260일, 일별 수익률 기준) ===
       N     p/T        등급
------------------------------
      50    0.04        양호
     100    0.08        양호
     200    0.16        양호
     300    0.24        양호
     400    0.32        양호
     500    0.40        양호

현재 전체 유니버스 평균 종목수 ≈ 408개 → p/T = 0.32 (양호)

[참고] 월별 T=60 기준이라면:
  N= 50: p/T=0.83  ← 주의
  N=100: p/T=1.67  ← 매우 불안정
  N=500: p/T=8.33  ← 매우 불안정
```

### 해석

| p/T 범위 | 등급 | 의미 |
|----------|------|------|
| < 0.5 | 양호 | Ledoit-Wolf 안정 수렴 |
| 0.5 ~ 1.0 | 주의 | 추정 분산 증가 |
| > 1.0 | 불안정 | singular 위험 |

**일별 Σ의 결정적 효과**: 월별 T=60이라면 N=50도 위험. 일별 T=1260으로 N=500도 양호.

### 설계 결정
**일별 수익률 + 패널 그대로 (월평균 ~410)** 채택.

### final과의 관계
- final/`bl_functions.py`의 `compute_sigma`도 일별 Ledoit-Wolf × 21 동일 사용
- final이 N=410으로 19개 실험 모두 안정 실행 → **p/T 진단 정확히 적중**
- final에 명시적 p/T 분석은 없음 → **04의 진짜 부가가치**

## Part 2 (Cell 5) — Prior 3-way 비교: mcap · 1/N · Risk Parity

**목적**: 세 가지 prior 후보가 만드는 균형수익률 π를 신호 단계에서 비교.

### 비교 대상

| Prior | 가중 방식 | 출발 가정 |
|---|---|---|
| `capm_mcap` | 시가총액 비례 | "시장이 정한 비중이 답" |
| `capm_eq` | 1/N 균등 | "정보 무시, 모든 종목 동등" |
| **`capm_rp`** | **1/σ 정규화** | **"위험 분산 최우선"** (Risk Parity) |

### Risk Parity 정의

```
w_i^RP = (1/σ_i) / Σ(1/σ_j)
```
→ 저변동 종목에 큰 비중. 위험 기여도(w_i × σ_i)가 모든 자산에서 동일 (자산 독립 가정).

### 실행 결과

```
=== Risk Parity Prior 추가 — 3-way 비교 수치 요약 ===

지표                          mcap        1/N    RP (1/σ)
HHI 평균                    0.01672    0.00268    0.00305
유효 종목 수 (1/HHI)              60        373        328
π 표준편차 평균               0.00375    0.00415    0.00370

=== π 상관계수 (3 pair) ===
mcap vs 1/N:  0.9865
mcap vs RP:   0.9867
1/N vs RP:    0.9981   ← 거의 완전 일치
```

### 핵심 해석

**(1) HHI 위치**: 1/N(0.00268) ≈ RP(0.00305) << mcap(0.01672)
- RP는 1/N에 매우 가까움
- mcap과 1/N·RP 사이 6배 차이 (메가캡 ~60개 집중 vs 350+개 분산)

**(2) π 상관계수 — 모든 pair가 0.98+**:
- mcap-1/N 0.9865 (이전 Part 2 발견)
- mcap-RP 0.9867 (RP는 mcap과도 거의 동등)
- **1/N-RP 0.9981** (RP는 1/N과 거의 완전 일치)
- → **세 prior 모두 신호 방향 거의 동일**, 차이는 *집중도*에만 있음

**(3) π 표준편차 — 셋이 거의 같음**:
- mcap 0.00375 / 1/N 0.00415 / RP 0.00370
- π = λΣw에서 Σ 변화가 prior 형태보다 더 큰 영향

### 4-패널 그래프 해석

**① Prior HHI 시계열** (log scale)
- 시간 무관 1/N과 RP는 0.0025~0.003 평탄 (거의 겹침)
- mcap은 0.010~0.021 사이 출렁임 (시장 집중도 따라)

**② π 표준편차 시계열**
- 세 prior 모두 거의 겹쳐 보임 (Σ 의존 우세)
- 2014, 2022년 spike (taper tantrum, 인플레이션 충격)

**③ π 상관계수 (3 pair) 시계열**
- 1/N vs RP: 항상 0.99~1.00 (가장 안정적)
- mcap vs 1/N, mcap vs RP: 0.97~1.00 (시점에 따라 약간 변동)

**④ 단일 시점 비중 분포** (저변동 50종목)
- mcap: 시총 따라 들쭉날쭉
- 1/N: 균등 (수평선)
- RP: 저변동 종목에 더 큰 비중 (감소형 곡선)

### final 백테스트로 검증됨

```
final/99_run 백테스트 (2010-2024):
  baseline (mcap):  Sharpe 1.106
  prior_eq (1/N):   Sharpe 1.105
  차이:             0.001  (통계적 노이즈)
```

→ **Part 2의 "세 prior 사실상 동등" 예측이 portfolio 단계에서 정확히 적중**.

## Part 3 (Cell 7) — 미니 BL 시뮬레이션 ⭐ NEW

**목적**: Part 2는 신호 단계(π) 비교. 본 절은 BL 전체 파이프라인을 직접 돌려 **사후 비중 w\*** 단계에서 prior 차이가 실제로 어떻게 나타나는지 측정.

### 시뮬레이션 절차

```
prior w  →  π = λΣw  →  μ_BL = π + τΣP^T(τPΣP^T+Ω)^-1(Q-Pπ)
                    →  optimize_portfolio (long-only)  →  w*
```

**시점**: 3개 대표 시점 (2013-10, 2017-07, 2024-01)
**P 행렬**: build_P (mcap 가중, 상하위 30%)
**Q**: 0.003 (월 0.3%, 저위험 우위 뷰)
**τ**: 0.1

### 실행 결과 — 사후 비중 통계

| 시점 | prior | max\|w\*\| | HHI(w\*) | 유효 종목 | top10 합 |
|---|---|---|---|---|---|
| **2013-10** | mcap | **8.08%** | 0.0356 | 28 | 49.6% |
| | 1/N | 4.77% | 0.0197 | 51 | 32.8% |
| | **RP** | **4.27%** | **0.0193** | **52** | **31.7%** |
| **2017-07** | mcap | **6.09%** | 0.0193 | 52 | 34.8% |
| | 1/N | 3.13% | 0.0095 | 106 | 19.5% |
| | **RP** | **2.80%** | 0.0096 | 104 | **18.6%** |
| **2024-01** | mcap | **6.39%** | 0.0241 | 42 | 39.3% |
| | **1/N** | **3.57%** | **0.0135** | **74** | **24.6%** |
| | RP | 3.96% | 0.0147 | 68 | 26.4% |

→ **mcap이 일관되게 가장 집중**. 1/N과 RP는 거의 동등.

### 핵심 발견 — w* 상관계수 (3시점 평균)

| Pair | π 상관 (신호) | **w\* 상관 (portfolio)** | 차이 |
|---|---|---|---|
| mcap vs 1/N | 0.9865 | **0.7484** | -0.24 ↓↓ |
| mcap vs RP | 0.9867 | **0.7497** | -0.24 ↓↓ |
| **1/N vs RP** | 0.9981 | **0.9834** | -0.01 (거의 변화 없음) |

### 결정적 인사이트

> **신호 단계에서 99% 일치하던 mcap과 1/N이 portfolio 단계에선 75%로 떨어짐 — MVO 최적화가 차이를 *증폭*함.**
>
> **반면 1/N과 RP는 portfolio 단계에서도 98% 일치 — 진짜 같은 portfolio 생성.**

### 시점별 mcap-1/N 상관 변화

```
2013-10: 0.835  (정상기)
2017-07: 0.807  (저변동)
2024-01: 0.603  ← AI 붐, 메가캡 쏠림 환경에서 더 큰 차이
```

→ AI 붐 같은 *메가캡 쏠림 환경*에서 mcap prior와 1/N의 차이가 더 커짐. 그러나 **1/N과 RP는 이 환경에서도 0.988로 거의 동일**.

### 의의

- 04 Part 2의 "신호 단계 거의 동일" 발견을 portfolio 단계로 확장 검증
- **prior 선택은 portfolio 단계에서 *의미 있게* 다른 결과 산출** (mcap vs 다른 두 개)
- **1/N과 RP는 portfolio 단계에서도 사실상 동등**

## 결론 — 04에서 가져갈 세 가지 핵심

### 1. p/T 진단 (Part 1)
- 일별 Σ + 21배 스케일링 → N=500까지 안정
- 월별 T=60은 N=50도 위험
- **final에 없는 분석. 04의 진짜 부가가치.**

### 2. Prior 3-way 비교 (Part 2)
- 신호 단계 π 상관: 모든 pair 0.98+
- **1/N과 RP는 거의 완전 일치 (0.9981)**
- HHI: 1/N ≈ RP << mcap (6배)
- final 결과(1.106 vs 1.105)가 정량 검증

### 3. 미니 BL 시뮬레이션 (Part 3) ⭐
- w\* 상관: mcap vs 1/N **0.75** (신호 0.99 → portfolio 0.75로 떨어짐)
- w\* 상관: 1/N vs RP **0.98** (portfolio 단계에서도 거의 동등)
- **MVO가 mcap 차이를 증폭, 1/N·RP 동등성은 유지**
- AI 붐(2024) 환경에서 mcap-1/N 차이 가장 큼 (0.603)

### Prior 권고 — 단계적

| 우선순위 | Prior | 근거 |
|---|---|---|
| **1순위** | **Risk Parity (RP)** | 본 프로젝트 가설과 가장 일관 + 실무 저변동 ETF 표준 (USMV·SPLV) |
| 2순위 | 1/N | 결과 거의 동일 (w\* 상관 0.98) + 더 단순 |
| 비추천 | 시가총액 | 메가캡 ~60개 집중, 가설과 무관 |

다만 EDA 결과 1/N과 RP의 portfolio 차이가 매우 작아(상관 0.98), **둘 중 어느 쪽을 써도 실질적으로 같은 결과**.

### 발표/리포트 권고 문구

> "**Part 2 (신호 단계)**: 세 prior(mcap·1/N·RP)의 π 상관계수 모두 0.98 이상. **1/N과 RP는 0.9981로 거의 완전 일치**.
>
> **Part 3 (portfolio 단계, 미니 BL)**: 신호 단계 99% 일치하던 mcap과 1/N이 portfolio 단계에선 75%로 떨어짐 (MVO가 차이 증폭). 반면 **1/N과 RP는 portfolio 단계에서도 98% 일치**.

> **결론**: 본 프로젝트는 저위험 ETF 컨셉으로 *Risk Parity prior가 이론적으로 가장 적합*하나, 1/N도 실질적으로 동등한 차선. 시가총액 prior는 가설과 무관하고 메가캡 쏠림으로 부적합.
>
> final 백테스트(baseline 1.106 vs prior_eq 1.105, 0.001 차이)가 04 신호 단계 발견을 정량 검증."
