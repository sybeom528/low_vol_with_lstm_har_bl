---

### 발견 1: GARCH — Q=0.001~0.003 구간이 Sharpe 고원

```
Q=0.001: Sharpe 0.922
Q=0.002: Sharpe 0.934  ← 최적
Q=0.003: Sharpe 0.931
Q=0.005: Sharpe 0.895  ← 급격 하락 시작
```

Q=0.001~0.003 구간은 Sharpe 차이가 0.012 이내로 평탄하다.  
최적 Q=0.002이지만, Q=0.003과의 차이(0.003)는 미미하여 **Q=0.001~0.003은 동등하게 로버스트한 선택**이다.

---

### 발견 2: Baseline — Q=0.003~0.010 구간이 넓은 고원 (Sharpe ≥ 0.999)

```
Q=0.003: Sharpe 0.999
Q=0.005: Sharpe 1.010  ← 최적
Q=0.007: Sharpe 1.007
Q=0.010: Sharpe 1.003
Q=0.015: Sharpe 0.945  ← 하락 시작
```

Baseline의 Sharpe 고원이 GARCH보다 훨씬 넓고(Q=0.003~0.010, 폭 0.011), 최고점(1.010)도 GARCH(0.934)보다 높다.

---

### 발견 3: Baseline이 GARCH보다 Sharpe 높음 (+0.076)

```
Baseline 최적 Sharpe: 1.010 (Q=0.005)
GARCH    최적 Sharpe: 0.934 (Q=0.002)
차이:                +0.076 (Baseline 우세)
```

**이 결과의 해석:**

| 관점 | 내용 |
|------|------|
| 포트폴리오 성과 | Baseline이 BL 입력 vol로 더 유리 → MVO 최적화 과정에서 직전 실현 vol이 다음달 포트폴리오 구성에 더 적합 |
| GARCH의 의미 | 04번에서 GARCH의 Rank IC(+0.0194 기여)는 **종목 분류(P 행렬)** 에서 의미 있음 — BL vol 입력으로서의 효과는 다름 |
| 미래 방향 | Baseline vol을 BL Σ 입력으로 쓰는 06번 전략이 08번(GARCH) 대비 유리할 수 있음 |

---

### 발견 4: Q 증가 → GARCH와 Baseline의 반응 패턴 차이

| Q 구간 | GARCH 패턴 | Baseline 패턴 |
|--------|-----------|-------------|
| 0.001~0.003 | Sharpe 0.922~0.934 (평탄) | Sharpe 0.957~0.999 (상승 중) |
| 0.003~0.007 | Sharpe 0.931→0.859 (하락) | Sharpe 0.999~1.010 (고원) |
| 0.007~0.010 | Sharpe 0.859→0.796 (급락) | Sharpe 1.007~1.003 (완만 하락) |
| 0.015~0.020 | Sharpe 0.699→0.619 (폭락) | Sharpe 0.945→0.902 (완만 하락) |

GARCH는 Q에 더 민감하게 반응하고, Baseline은 넓은 범위에서 안정적 성과를 유지한다.

### 발견 5: MDD 특성 차이

```
GARCH  최저 MDD: Q=0.005에서 -17.87% (수익률도 낮은 트레이드오프)
         최고 MDD: Q=0.020에서 -26.49%
```

Baseline은 Q가 클수록 MDD가 개선되는 경향이 있다 (Q=0.015까지). 이는 GARCH와 반대 패턴.  
다만 Q=0.015부터 Sharpe가 급락하므로 MDD만 보고 큰 Q를 선택하는 것은 부적절하다.

## Part 4. Q와 BL 공식의 연결

$$\mu_{BL} = \pi + \tau\Sigma P^T (P\tau\Sigma P^T + \Omega)^{-1} \underbrace{(Q - P\pi)}_{\text{조정 크기}}$$

- `Pπ` ≈ 음수 (CAPM은 고위험 선호)
- `Q - Pπ` = Q + |Pπ| → Q가 클수록 조정 크기 ↑
- 조정이 과도해지면 저위험 종목에 극단적 집중 → 분산 효과 손실

| Q 값 | BL 조정 강도 | 포트폴리오 성격 |
|------|------------|----------------|
| 0.001~0.003 | 약함 | CAPM에서 소폭 이탈, 분산 유지 |
| 0.005~0.007 | 중간 | 저위험 tilting 가시적 |
| 0.010~0.020 | 강함 | 과도한 집중, 분산 효과 손실 |
