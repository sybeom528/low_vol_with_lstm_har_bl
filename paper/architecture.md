3. 방법론
   1. 3.1 변동성 예측 모델(사용자 본인의 논문 작성 담당 파트)
      1. 3.1.1 종속변수, 입력변수, BL 입력 변환

         - LSTM(장기 기억에 강점)- HAR(금융 표준 변동성 모형)

   2. 3.2 블랙-리터만
      1. 3.2.1 블랙-리터만 모델 및 제안 슬롯- BL 수식- 각 구조에 제안 슬롯에 대한 수식과 설명(q_lambda 등)
      2. 3.2.2 Walk-Forward 구조S&P 500에 포함된 종목의 변동성 예측 후 포트폴리오 비중 산출
      3. 3.2.3 평가 방법- 레짐 분석 Appendix. 레짐 분석 결과와 금융 도메인 기반으로 4개 구간으로 나눔.
4. 실증 분석
   1. 4.1 변동성 예측 결과- 전 구간과 각 레짐에서 LSTM이 ANN 대비 모두 우수한 성능(RMSE, Spearman, Hit rate)- 특히 R3(금융위기)구간에서 성능 차이가 확연하게 드러남. 이게 결과로 어떻게 전이되는지는 5.2에서 확인 
   2. 4.2 Baseline Slot result- Pyo&Lee(2018)의 제안 슬롯을 한 가지씩만 변화시키면서 성능 비교- Q행렬 양수 시 금융위기 구간에서 안정적인 성능(fpm 대비)- P행렬 동일 가중, 역변동성 가중으로 구성 시 변동성 모델의 sharpe 증가- omega 역시 he 표준 대비 성능 증가
   3. 4.3 P행렬에 따른 결과- P행렬 3가지를 비교 및 omega를 err로 고정해서 비교- marginal box plot(전체, 레짐별)- P행렬에 따른 각 레짐별 평균 수치와 LSTM 우위 표- P행렬을 eq,rp로 구성하기만 하더라도 전체 sharpe 증가- R3 구간은 슬롯 관계 없이 LSTM이 ANN 대비 포트폴리오 성과가 좋음 → 금융위기 구간에서 상승장에서 회복한 효과- R4 구간 fpm의 성능 우위(다만, P행렬 mcap 구성 시 고변동 전략 변화. eq, rp 구성 시 완화)- 벤치마크 대비 성능(여기는 아직 내가 다 생각 몬함)
5. 결론 및 한계
   - 블랙-리터만에서의 뷰는 보통 수익률로 구성 → 변동성 예측과 저변동 포트폴리오 결합을 위해서는, 적절한 슬롯 구성이 필요
   - P행렬을 mcap로 구성해서 시가총액 비중으로 구성하는 것 보다, 동일가중이나 역변동성 가중이 효과적
   - 금융위기 구간에서는 Q 양수 설정의 효과가 드러남
   - 제안한 앙상블 모델의 성능이 ANN 대비해선 좋지만, hit rate 고려 시 충분히 좋다고 할 수 없음(LSTM 모델 고도화(입력 피처 충분히, multi layer 등)해서 향후 연구에 반영)
   - R1 구간 성능 ANN 대비 하락(결과 나온 이유 Appendix)
   - 결과에 대한 해석이고, 인과관계는 확인하지 못 함. 향후 연구에 반영

## 본 논문의 목적
### 비교
1. 우리 방식(모델)이 변동성을 더 잘 예측함.
2. 슬롯 구성을 살릴 수 있는 구조임.
### 저변동 포트폴리오 전략의 이점
1. 변동성 예측을 통해 저변동 포트폴리오 전략 모델을 구축하였다.
2. 최근 AI랠리(우리 분석의 레짐 중 R4)구간을 제외하더라도 높은 Sharpe, Sortino 값을 낼 수 있었으며, 수익률 또한 좋은 모습을 보여준다.
3. 특히 코로나 위기 + 회복기(우리 분석의 레짐 중 R3)에서 특히 상대적으로 강한 모습을 보여준다.
