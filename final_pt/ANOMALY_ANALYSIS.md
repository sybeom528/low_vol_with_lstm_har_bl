## 02b 검증 목표

S&P500 유니버스에서 **변동성 기준 분위 포트폴리오**를 구성하고,
저변동성(Q1)이 고변동성(Q5) 대비 위험조정 수익이 높은지 검증한다.

---

## 핵심 결론 (forward portfolio sort, 2010~2025)

| 질문 | 결과 |
|---|---|
| **sort 변수 결정** — 저베타 vs 저변동 Q1 비교 | ✓ 저변동 Sharpe 0.986 > 저베타 0.938, 누적 581% > 512% |
| 30%/30% Sharpe 우위 | ✓ Q1 0.955 vs Q5 0.727 |
| 30%/30% MDD 우위 (하방 위험) | ✓ Q1 −16.7% vs Q5 −34.1% (고변동 손실폭 2배) |
| 임계값 강건성 (20~40%) | ✓ 모든 임계값에서 Sharpe·MDD 격차 양수 → 30% 선택 robust |
| 시간 일관성 (2-year 구간별 위험대비효율) | ✓ 평균 Sharpe 격차 +0.301, 평균 MDD 격차 +10.7%p |

**이상현상의 정확한 의미**: "저변동이 더 번다"가 아니라 "**위험 1단위당 수익이 더 좋다 + 위기 구간 손실폭을 절반으로 줄인다**".
저변동 우위의 본질은 **Sharpe 와 MDD 격차** — Q5 고변동의 −34% drawdown 은 실투자에서 견딜 수 없는 수준이며, 이것이 BL 의 공분산 기반 reweight 로 고변동 비중을 자동 억제해야 하는 이유.

---

## 방법론

### 포트폴리오 정렬 (Portfolio Sort, forward 형식)
- 매월 말 `vol_252d` (252일 실현변동성) 기준 5분위 정렬
- Q1: 저변동성, Q5: 고변동성
- 각 분위 동일가중 수익률 계산
- **Look-ahead bias 방지**: 당월 말 t의 vol/beta 로 분류 → **t+1 (다음 달) 수익률 측정** (`fwd_ret_1m` 사용)
- **Realization date indexing**: 결과 시리즈 인덱스 `+ MonthEnd(1)` shift → SPY/rf 와 자연 정렬 (Frazzini-Pedersen 2014 표준)
- **생존편향 완화**: `sp500_membership` 기반 해당 월 편입 종목만 포함

### Section 2. 30%/30% 분류 기준 + 임계값 민감도

**목적**:
1. BL P 행렬 기준(저변동 30% / 고변동 30%)으로 Sharpe·MDD 비교
2. 임계값을 20%~40% 범위로 바꿔도 저변동 우위 방향이 유지되는지 (민감도 분석) → **30% 선택의 robust 근거**

| 그룹 | 연환산 수익 | 연환산 변동성 | Sharpe | MDD | 누적 수익 |
|---|---|---|---|---|---|
| 저변동 (하위 30%) | 12.79% | 11.75% | **0.955** | **-16.74%** | 587.13% |
| 고변동 (상위 30%) | 18.29% | 23.01% | 0.727 | -34.05% | 1108.55% |

Sharpe 우위 (+0.228) + MDD 우위 (저변동 손실폭이 고변동의 절반) — **BL P 행렬 구조의 데이터 근거**.

## 종합 해석 (forward portfolio sort, 2010~2025)

| 검증 항목 | 방법 | 결과 |
|---|---|---|
| sort 변수 선택 (저베타 vs 저변동) | Q1 직접 비교 (Section 1) | 저변동 Sharpe 0.986 > 저베타 0.938, 누적 581% > 512% ✓ |
| 30%/30% Sharpe·MDD 비교 | Section 2 | Q1 0.955 vs Q5 0.727, MDD −16.7% vs −34.1% ✓ |
| 임계값 강건성 (20~40%) | Section 2 민감도 | 5개 임계값 모두 Sharpe·MDD 격차 양수 → 30% robust ✓ |
| 위험대비효율 시간 일관성 | Section 3 (2-year 구간별) | 평균 Sharpe 격차 +0.301, 평균 MDD 격차 +10.7%p ✓ |

**핵심 결론**:
- **저변동 > 저베타**: 같은 "저위험" 분류라도 vol_252d 기준이 Sharpe·누적수익 모두 우위 → 변동성을 sort 변수로 채택하는 근거.
- **MDD 격차가 결정타**: Q5 −34.1% 의 손실폭은 실투자에서 견디기 불가능한 수준. 50% 손실에서 회복하려면 100% 상승이 필요 → raw 누적수익이 아무리 높아도 이 drawdown 을 통과하지 못하면 자본 보전 실패.
- **임계값·시간 무관 robust**: 20~40% 어느 컷오프, 2-year 구간 어느 시점에서도 위험대비효율 우위 유지.

## BL 모델 연결

저변동 우위의 가치는 **"평균 수익률을 더 잘 맞추기 때문"** 이 아니라 **"위기 구간 손실폭을 절반으로 줄이기 때문"** 에 있다. 이 점이 BL framework 채택의 직접적 근거:

| 분석 결과 | BL 설계 함의 |
|---|---|
| 저변동 vs 저베타 Sharpe 우위 | sort 변수를 vol_252d 로 선택 (BL P 행렬의 분류 기준) |
| Q1 Sharpe 0.986 vs Q5 0.704 | P 행렬 방향(저변동 long / 고변동 short) 정당화 |
| 30%/30% 격차 (Sharpe +0.23, MDD +17.3%) | P = [+1/n_저변동, −1/n_고변동] 구조 정당화 |
| Q5 MDD −34% (실투자 견딜 수 없음) | 공분산 Σ 로 고변동 비중 자동 억제 → MDD 방어 |
| 임계값 강건성 (20~40%) | q_value/PCT_GROUP 민감도 분석 (99_analyze §M/§N) 의 데이터 근거 |

**왜 단순 MVO 가 아니라 BL 인가**:
1. **MVO 만으로는 부족**: 단순 평균수익 + 공분산 입력으로 MVO 를 풀면, raw return 이 높은 고변동 자산이 과대 배분됨 → MDD −34% 같은 충격에 노출.
2. **자산별 μ 직접 예측 불가**: XGBoost·TabPFN 등 모든 시도 R²≈−0.95 (학계 합의).
3. **BL 의 해법**: prior π (CAPM 균형) 에 단일 spread view (저변동 > 고변동, *risk-adjusted 기준*) 를 합성. 공분산 Σ 가 자동으로 고변동 자산 비중을 억제 → MDD 방어와 평균 우위를 동시 달성.
4. **Prior 다양화**: 본 프로젝트는 prior 슬롯에 mcap·eq·rp 세 가지 옵션을 모두 매트릭스로 실험 → 단일 prior 가정의 위험을 회피.

## 한계

1. **고변동 그룹의 견딜 수 없는 drawdown**: Q5 MDD −34.1% 는 raw 누적수익이 우위라도 실투자에서 청산 트리거 가능성이 높음. 본 프로젝트가 BL 을 채택한 직접적 동기 — 공분산 행렬로 고변동 비중을 억제해 MDD 를 방어.
2. **후향적 변동성**: `vol_252d`는 과거 1년 실현 변동성 → 미래 분류에 오류 가능성 존재. 03b LSTM 이 **forward 21d log-RV** 를 직접 예측해 이 한계를 보완.
3. **delisting 후 데이터 미수집**: 시점별 S&P500 멤버십 필터로 편출 종목도 편출 시점까지는 패널에 포함되지만, 파산·인수 등으로 상장폐지된 이후의 수익률(주로 큰 음수)은 yfinance에서 수집되지 않아 `daily_returns.pkl` 에 부재 → 고변동 그룹의 잔존편향 가능성 (실제 손실보다 양호하게 추정될 가능성).
4. **기간 의존성**: 이상현상 강도가 레짐(장세)에 따라 변함 — QE 강세장(2010~2015)에서 약화, 2005~2009 GFC 구간에서 명확.
